# Terminal

This document explains how to use the `sec5` package from the terminal to enhance your workspace security effortlessly.

## Installation

Before using the `sec5` package, you need to ensure that you have Node.js and npm installed on your system. Then, you can install the package globally using npm:

```bash
$ npm install -g sec5
```

## Usage

Once the package is installed, you can run it from the terminal using the following command:

```bash
$ sec5
```

This command will execute the `sec5` package and perform various security checks on your workspace.

## Example Output

After running the `sec5` command, you might see output similar to the following:

```
[SEC5 WARN] Found 2 comments in:
C:\Users\k4dir\Desktop\Projeler\sec5\bin\sec5.js
C:\Users\k4dir\Desktop\Projeler\sec5\src\todoCheck.ts

[SEC5 INFO] CHANGELOG.md and package.json version matches
[SEC5 INFO] Codes are formatted correctly with Prettier
```

This output provides information about any security warnings or information gathered during the security check process.

## Conclusion

Using the sec5 package from the terminal is a straightforward way to enhance your workspace security. By running the command and reviewing the output, you can identify potential security vulnerabilities and ensure that your workspace is safe and secure.
