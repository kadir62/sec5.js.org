# Sec5.js Documentation

Sec5 uses [Nextra](https://nextra.site) for documentation.

## Local Development
To run the project locally, first install the dependencies (`pnpm` recommended):
```bash
pnpm i
```

Next, run the development server:
```bash
pnpm dev
```
